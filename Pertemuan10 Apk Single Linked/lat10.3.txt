Penerapan Aplikasi Stack dengan Array

Stack dapat diimplementasikan menggunakan Array, di mana setiap elemen array menyimpan satu data dari tumpukan.
Kita juga menggunakan variabel TOP untuk menandai posisi elemen teratas.

Berikut prinsip kerjanya:

Inisialisasi Stack:

Set nilai TOP = -1 menandakan stack masih kosong.

Operasi PUSH (menambah data):

Sebelum menambah, periksa apakah stack sudah penuh (TOP == MAX-1).

Jika belum, tambahkan TOP = TOP + 1, lalu simpan data ke STACK[TOP].

Operasi POP (menghapus data):

Periksa apakah stack kosong (TOP == -1).

Jika tidak kosong, ambil data di STACK[TOP], lalu kurangi TOP = TOP - 1.

Operasi PEEK (melihat data teratas):

Menampilkan isi STACK[TOP] tanpa menghapusnya.
