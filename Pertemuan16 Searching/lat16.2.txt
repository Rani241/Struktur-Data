BinarySearch(A, N)
1.  Set low ← 0
2.  Set high ← panjang(A) - 1

3.  Selama low ≤ high lakukan:
4.      mid ← (low + high) div 2

5.      Jika A[mid] = N maka
6.          RETURN "Data ditemukan pada index mid"

7.      Jika N < A[mid] maka
8.          high ← mid - 1
9.      Jika tidak maka
10.         low ← mid + 1

11. RETURN "Data tidak ditemukan"
