a. inisialisasi
DEQ dibuat kosong
FRONT = -1
REAR = -1

b. Insert dari kanan
jika REAR == MAX-1 → antrian penuh kanan
jika FRONT == -1 → FRONT = 0
REAR = REAR + 1
DEQ[REAR] = data baru

c.Insert dari kiri
jika FRONT == 0 → antrian penuh kiri
jika FRONT == -1 → FRONT = 0, REAR = 0
lainnya → FRONT = FRONT - 1
DEQ[FRONT] = data baru

d.Delete dari kanan
jika FRONT == -1 → antrian kosong
hapus DEQ[REAR]
jika FRONT == REAR → FRONT = REAR = -1
lainnya → REAR = REAR - 1
