a. Menghitung dan mencetak jumlah simpul

Gunakan perulangan dari simpul pertama (FIRST) sampai terakhir (NULL) sambil menambah penghitung.
count = 0
Q = FIRST
selama (Q != NULL):
    count = count + 1
    Q = Q->LINK
cetak "Jumlah simpul =", count


b. Menghitung dan mencetak TotalINFO
total = 0
Q = FIRST
selama (Q != NULL):
    total = total + Q->INFO
    Q = Q->LINK
cetak "Total INFO =", total


c. Mencetak semua nilai INFO ke layar
Q = FIRST
selama (Q != NULL):
    cetak Q->INFO
    Q = Q->LINK


d. Mencetak jumlah simpul yang nilai INFOnya = 50
jumlah = 0
Q = FIRST
selama (Q != NULL):
    jika (Q->INFO == 50):
        jumlah = jumlah + 1
    Q = Q->LINK
cetak "Jumlah simpul dengan INFO = 50 =", jumlah

Hasil akhirnya adalah berapa simpul yang memiliki nilai 50.
