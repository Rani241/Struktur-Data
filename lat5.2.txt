Tulislah Algoritma Dasar dari:  
a. Mengisi Stack1 (PUSH1) 
Algoritma PUSH1 
Deklarasi: 
data : integer 
Langkah-langkah: 
1.  Jika (top1 + 1 == top2) maka 
Tulis "Stack penuh!" 
2.  Jika tidak, maka 
top1 ← top1 + 1 
Stack[top1] ← data 
Tulis "Data berhasil dimasukkan ke Stack 1" 
Selesai 

b. Menghapus isi Stack1 (POP1)  
Algoritma POP1 
Deklarasi: 
data : integer 
Langkah-langkah: 
1.  Jika (top1 == -1) maka 
Tulis "Stack 1 kosong!" 
2.  Jika tidak, maka 
data ← Stack[top1] 
top1 ← top1 - 1 
Tulis "Data yang dihapus: ", data 
Selesai 

c. Mengisi Stack2 (PUSH2)  
Algoritma PUSH2 
Deklarasi: 
data : integer 
Langkah-langkah: 
1.  Jika (top2 - 1 == top1) maka 
Tulis "Stack penuh!" 
2.  Jika tidak, maka 
top2 ← top2 - 1 
Stack[top2] ← data 
Tulis "Data berhasil dimasukkan ke Stack 2" 
Selesai 

d. Menghapus isi Stack2 (POP2) 
Algoritma POP2 
Deklarasi: 
data : integer 
Langkah-langkah: 
1.  Jika (top2 == ukuran_array) maka 
Tulis "Stack 2 kosong!" 
2.  Jika tidak, maka 
data ← Stack[top2] 
top2 ← top2 + 1
