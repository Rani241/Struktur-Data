Tulislah algoritma yang lengkap untuk :  
a. Mengisi Stack1 (PUSH1)  
Algoritma PUSH1 
1. Jika (top1 + 1 == top2) maka 
2. Tulis "Stack penuh!" 
3. Jika tidak, maka 
4. top1 ← top1 + 1 
5. Stack[top1] ← data 
6. Tulis "Data berhasil dimasukkan ke Stack 1" 
Selesai 

b. Menghapus isi Stack1 (POP1)  
Algoritma POP1 
1.  Jika (top1 == -1) maka 
2.  Tulis "Stack 1 kosong!" 
3.  Jika tidak, maka 
4.  data ← Stack[top1] 
5.  top1 ← top1 - 1 
6.  Tulis "Data yang dihapus: ", data 
Selesai 

c. Mengisi Stack2 (PUSH2)  
Algoritma PUSH2 
1.  Jika (top2 - 1 == top1) maka 
2.  Tulis "Stack penuh!" 
3.  Jika tidak, maka 
4.  top2 ← top2 - 1 
5.  Stack[top2] ← data 
6.  Tulis "Data berhasil dimasukkan ke Stack 2" 
Selesai 

d. Menghapus isi Stack2 (POP2) 
Algoritma POP2 
1.  Jika (top2 == ukuran_array) maka 
2.  Tulis "Stack 2 kosong!" 
3.  Jika tidak, maka 
4.  data ← Stack[top2] 
5.  top2 ← top2 + 1 
6.  Tulis "Data yang dihapus: ", data 
Selesai
